# TI-Basic Emulator

A simple TI-Basic parser and interpreter that is capable of running some programs from my old TI-83+ calculator. Since I was mostly interested in running these specific programs, only a subset of the language is implemented so YMMV.

<img src="https://user-images.githubusercontent.com/160978/212590865-1f1f4bfc-41ec-4645-b00b-69a827817f30.jpg" width="50%">

## Running

```
npm run start
```
